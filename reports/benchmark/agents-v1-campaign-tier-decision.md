# DiscoveryBench Agents v1 campaign-tier decision

Decision: `sealed-pilot-ready-owner-authorization-pending`.

The next authorizable tier is one sealed 50-task engineering batch, two exact
cloud snapshots, one repeat, five architectures, and exact comparators. It is
non-inferential and cannot promote a claim. Its purpose is to prove custody,
trace volume, parsers, structured-output behavior, error accounting, token/cost
measurement, and contamination probes.

The 200-task base remains the target claim-grade design but is not the next
executable tier because the audited host cannot run the registered open/local
candidate. Public calibration, full-cell, and high-sensitivity tiers remain
separate. No tier is authorized.
