# Optionality portfolio audit

The registry contains nine entries: five workstreams, three unregistered
theorem gates, and Reliable Discovery. Every entry records readiness, residual
value, decay, expiry horizon, trigger, first irreversible spend, kill/defer
condition, required evidence, AI-native validation, dependencies, current
authority, prohibitions, visibility, and next review date.

DiscoveryBench Agents is uniquely first to exercise. All other options remain
design, draft-standard, theorem-gate, or preserved-deferred entries. No value
is a scientific score, expected-value estimate, measured impact, execution
promise, or permission for parallel substantive branches. Stopped and deferred
options remain visible.
