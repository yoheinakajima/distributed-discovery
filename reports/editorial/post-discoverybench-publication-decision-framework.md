# Post-benchmark publication decision framework

This framework applies only after a claim-grade Agents v1 campaign. It creates
no benchmark paper, title-level result, provider ranking, lifecycle change, or
publication authority.

The future formal instrument name is TreasureBench. The prospective working
title is *TreasureBench: A Benchmark for Collective Search under Shared and
Private Evidence*. Treasure Hunt must appear in the first section only as the
playable companion. Any formal benchmark paper must distinguish this evidence-
and-action-portfolio instrument from classical mobile-agent treasure-hunt
navigation using primary sources.

| Evidence pattern | Candidate response |
| --- | --- |
| Common cross-family agent pattern | Grand sequel plus technical companions |
| Heterogeneous family effects | Standalone instrument-and-results paper while preserving modular theorem papers |
| Theory-transfer failure | Methods/instrument paper with nulls and an explicit transfer boundary |
| Strong instrument, weak or unstable effects | Methods/instrument paper with nulls |
| One or two dominant theorem families | Standalone instrument/results paper plus targeted companions |
| Major contamination or validity limit | No paper; preserve the negative result and audit |

Every branch preserves primary claim ownership. A future lifecycle action
requires a separate evidence-backed decision; this framework authorizes no
archive, supersession, or paper.
