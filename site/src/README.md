# Site source

The live research library is generated by
`src/distributed_discovery/site/build.py`. `styles.css`, `site.js`, and `og.png`
are copied into every build; substantive numbers, study statements, claims,
runs, and publication metadata continue to come from validated repository
sources.

The five HTML files in this directory are retained legacy companion-page
prototypes. They are not imported by the current generator and must not be
treated as the live page shell. Public presentation copy is mapped in
`design/site-refresh/copy-map.yml`, while generated HTML remains complete
without JavaScript. GitHub Actions deploys the ignored `site/dist` artifact from
`main`.
