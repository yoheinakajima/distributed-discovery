# Institutions for Distributed Discovery

Working paper subtitle: *Evidence Acquisition, Adaptation, Coverage, Incentives,
and Measurement*. This paper synthesizes completed bounded studies; it neither
replaces the canonical paper nor claims a paper-wide evidence status. Its build
will regenerate tables/figures from the checksum-validated DD-004--DD-008 runs.

The Discovery Stack is an organizing taxonomy: acquire → disclose → allocate →
adapt → cover → reward → measure. It is not an empirical theorem.
