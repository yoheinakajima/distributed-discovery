# Distributed Discovery with Private Information

Working subtitle: **The Value of Roles Without Communication**.

This is a structured working draft, not a polished theorem paper. The supported content currently includes the finite model, deterministic-sufficiency proof, benchmark inequalities, an exact 21-point tiny grid, informative counterexamples to general clue-following optimality, the lossless policy-signature/feasibility theorem, an independent signature reproduction, and the exact canonical signature state-space barrier.

The canonical global optimum remains unresolved: direct clue-following is a certified lower bound and the pooled planner is only an external numerical upper benchmark. Do not describe the signature reduction as an objective certificate. The next draft revision is the analytic \(N=2\) threshold result.
