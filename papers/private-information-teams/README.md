# Distributed Discovery with Private Information

Working subtitle: **The Value of Roles Without Communication**.

This is a structured working draft, not a polished theorem paper. The verified content currently supports: the finite model, deterministic-sufficiency proof, benchmark inequalities, an exact 21-point tiny grid, informative counterexamples to general clue-following optimality, and a canonical lower-bound search with unresolved global optimum.

Do not promote the canonical observation to a theorem. The next draft revision depends on analytic \(N=2\) thresholds and a certified upper-bound method.
