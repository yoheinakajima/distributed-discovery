# The Common-Source Trap

Working paper: *The Common-Source Trap: Endogenous Independent Evidence in
Distributed Discovery*.

Build with `make common-source-trap`. The builder validates every source artifact
against its immutable run manifest, regenerates all figures and tables, resolves
claim IDs and citations, compiles twice with a fixed source epoch, and requires a
20–32 page byte-reproducible PDF. The checked PDF is a working paper, not a
submission, peer-reviewed publication, DOI artifact, or empirical study.
