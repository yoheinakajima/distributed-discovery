# The Incentive to Ignore

Working paper: *The Incentive to Ignore: Selective Attention and Audience
Design in Distributed Discovery*.

Build with `make incentive-to-ignore`. The builder checksum-validates DD-012
through DD-014 and DD-011 sources, regenerates nine evidence assets, resolves
claims and citations, compiles twice with a fixed source epoch, and requires a
byte-reproducible 20–32 page PDF. This is a working paper with no DOI; it is not
submitted or peer reviewed, and it reports no human experiment.
