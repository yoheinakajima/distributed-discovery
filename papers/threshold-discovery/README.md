# Threshold Discovery

Working paper: *Threshold Discovery: Coordination, Crowding, and Minimum
Viable Teams*.

Build with `make threshold-discovery`. The builder checksum-validates DD-016,
DD-017, both separately registered DD-015 runs, DD-018, DiscoveryBench v3, and
Synthetic Experiment v3; regenerates eight evidence assets; resolves claim and
citation keys; compiles twice with a fixed source epoch; and requires a
byte-reproducible 20-32 page PDF. This is a working paper with no DOI. It is not
submitted or peer reviewed, makes no verified novelty claim, and reports no
human experiment.
