# Artifact license matrix

The repository root is MIT-licensed. That evidence covers original repository
code and materials, but a release or submission still needs artifact-specific
review. This audit changes no license and grants no rights beyond the existing
license and applicable third-party terms.

| Artifact class | Current position | Release/submission gate |
| --- | --- | --- |
| Code, data, schemas, fixtures | Root MIT license | Preserve copyright and license notice |
| Site code and prose | Root MIT license | Distinguish presentation from manuscript rights |
| Paper source and PDFs | Root MIT license | Owner selects current arXiv terms and checks any later venue transfer |
| Generated figures | MIT when wholly original | Confirm provenance and embedded assets |
| Pinned upstream material | Upstream MIT, read-only here | Cite the pin; do not transfer or relabel upstream rights |
| Third-party citations/assets | Third-party terms control | Redistribute only metadata, lawful quotation, or separately permitted assets |

The machine-readable matrix records arXiv suitability, Zenodo suitability,
copyright-transfer risk, evidence, and remaining owner decisions for every
class. The unresolved decisions block an archival release or submission but do
not change current repository use.
