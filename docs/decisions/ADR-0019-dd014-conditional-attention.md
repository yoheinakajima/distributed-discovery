# ADR-0019: Bound DD-014 to a proved conditional policy class

Status: accepted

Date: 2026-07-21

## Decision

DD-014's main result covers the complete deterministic, agreement-respecting,
label-equivariant disagreement class for three labels: private-dominant,
public-dominant, and third-option contrarian. It must not be described as an
unrestricted deterministic-policy theorem.

A separate two-agent, two-label audit enumerates all sixteen raw deterministic
policy tables. Its results are adversarial scope evidence and remain separate
from the main census and theorem.

## Consequences

The anonymous reduction is lossless within its declared class. The raw audit
can reveal policies—such as complementary constant actions—that invalidate
broader interpretations. Any unrestricted extension needs a new registration,
proof boundary, and immutable run.
