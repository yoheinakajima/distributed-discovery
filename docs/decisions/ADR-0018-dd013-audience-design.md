# ADR-0018: DD-013 audience design and information firewalls

DD-013 inherits DD-012's exact three-target clue model and rational grid. It
separates four institutions: binding signal delivery, voluntary ex-ante action-
mode choice by recipients, public access plus binding action assignment, and
symmetric signal garbling. An audience size is therefore not silently treated
as either realized attention or enforceable action.

The primary census contains 175 `(N,p,q)` cells, 1,050 binding audience rows,
4,025 voluntary audience/profile rows, and 2,625 feasible garbling rows. The
garbling family is deliberately bounded to symmetric delivered accuracies from
the registered grid satisfying `1/3<=g<=q`. No arbitrary experiment or
Blackwell-order result is claimed.

Eight firewall forms record access, action permission, access verification,
action verification, commitment, and subsidy assumptions. Exact implementation
is evaluated for binding exclusive delivery and public universal pooling; the
latter is ex-post budget balanced against the unit success prize but remains
scoped to the registered ex-ante two-mode deviation class.

Closed forms are primary. A separate target/shared/private-signal enumerator
must reproduce discovery, action quality, distinct actions, voluntary
deviations, pooling incentives, and garbling comparisons across 3,319,200
unique labeled states. Four targeted corruptions must be rejected. The run has
a 12-minute/2-GB hard cap and no valid partial claim outcome.
