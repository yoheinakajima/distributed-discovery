## What is specific to the current framework

The following appear to be project-specific **organizational choices**, not established novelty claims:

- using one tuple to index evidence, information, actions, coverage, budget, timing, and rewards;
- treating the Shared Discovery Paradox as the atomic motivating benchmark;
- using the institutional 2×2 matrix to separate private/pooled evidence from autonomous/assigned action;
- using “information frontier minus protocol loss” as a shared accounting vocabulary across registered studies;
- maintaining claim/run provenance as part of the research program.

Whether these choices constitute a novel theoretical contribution remains unassessed. The decomposition itself is definitional.

## Terminology collisions

- **Distributed discovery** already describes autonomous-science agent coordination (Wang et al., 2026), resource/service discovery, distributed data-dependency discovery, and news discovery. Always attach the project definition.
- **Information frontier** has broad informal uses and may evoke production/possibility frontiers; define it mathematically each time.
- **Protocol loss** is generic and appears in engineering contexts; always name the benchmark frontier.
- **Effective channels** can mean effective sample size, rank, capacity, or independent sources. The upstream count is model-specific.
- **Action redundancy** may be beneficial robustness in robotics/coverage; avoid equating all repetition with waste.
- **Price of anarchy** and **information design** are established terms and should retain their standard scope.

## Softer wording required

- Say “research program,” “framework,” or “research subject,” not “new field.”
- Say “working definition” and “project vocabulary,” not “first definition.”
- Say “canonical computational result” for the copying crossover, not a general theorem.
- Say “verified upstream execution” separately from “independent reproduction.”
- Describe applications as mappings with changed primitives, not direct empirical validation.
