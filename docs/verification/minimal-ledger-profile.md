# Minimal claim-ledger verification profile

This draft defines a tool-neutral minimum for machine-readable claim
provenance and validation. It is an optional interoperability profile, not a
standard, certification, or research-quality score.

A conforming record identifies a claim, statement, scope, evidence category,
source reference, proof/certificate/run reference where applicable, workflow
status, checksum or provenance object, validation result, and preserved
negative or failure information where relevant.

Conformance means only that the record is structurally complete and validates
against the draft schema. It does not certify scientific quality, novelty,
correctness, completeness, priority, reproducibility in practice, or peer
review. It cannot replace domain review or proof.

The included fixtures are synthetic. Any future exercise must first use an
unrelated synthetic or AI-generated fixture repository. No external human
project or adoption claim is authorized.
