# Institutional matrix — DD-000

| Evidence / action authority | Autonomous or unassigned | Coordinated or assigned |
|---|---|---|
| Local/private evidence | Private clue-following | Private-team optimum |
| Pooled/shared evidence | Consensus or market | Planner portfolio |

The columns isolate action assignment only when the information row is fixed; the rows isolate sharing only when action authority is fixed. The canonical planner-versus-private comparison changes both information access and assignment authority, so it does not isolate the value of information. DD-001 studies the missing private-team cell.
