# Frozen baseline model

There are three uniformly likely targets and `N in {2,...,8}` agents. Each has
a conditionally independent private clue of accuracy `p`; one shared clue has
accuracy `q`, with incorrect labels uniform. An ex-ante access gate selects
`k` attending roles. Attenders receive the shared clue and search its label;
nonattenders do not receive it and search their private-clue labels. Successful
agents split a unit prize equally and social value is union discovery.

The registered extension will use exact rational arithmetic and will state any
additional reward and observability technologies separately.
