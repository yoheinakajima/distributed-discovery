# Frozen model

The model is exactly DD-008A's `M=3` uniform-target source-choice game, now with
arbitrary finite `N>=2` and `0<p<1`. A free common source gives every `C` user
the same conditionally `p`-accurate signal; each costly `I` source gives its user
an independent conditionally `p`-accurate signal. Agents search their signals
directly and split a unit prize equally when correct. Social net value is union
discovery less `c` per independent source.

No disclosure, allocation choice, communication, learning, heterogeneous
accuracy, or real data enter this analytic gate.
