# Draft researcher protocol — deployment prohibited

1. Obtain ethics, institutional, security, payment, and accessibility approvals.
2. Freeze the public preregistration and code commit before assignment.
3. Generate the signed assignment manifest offline; conceal future assignments.
4. Use only approved consent language and synthetic dry runs until authorization.
5. Record protocol deviations without editing the frozen hypotheses.
6. Apply the fixed stopping rule; do not inspect outcomes to extend enrollment.
7. Separate identity/payment operations from research records.
8. Publish exclusions, missingness, null results, and deviations.

Steps 1–8 are a proposal. This repository performs none of the live operations.
