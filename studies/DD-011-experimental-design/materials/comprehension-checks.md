# Draft comprehension checks

1. When is an independent-source cost charged? **When the source is selected or assigned, before its signal is known.**
2. Can a public report reveal another participant's private clue before reporting? **No.**
3. Under marginal coverage, does duplicating an already covered correct action earn the unique-coverage component? **No.**
4. Does a visible prior action guarantee it is correct? **No.**
5. May a participant withdraw? **Yes, subject to the approved consent and payment rules.**

The proposed manipulation check requires all five correct before continuing, with
one neutral reread opportunity. This rule is a draft, not a deployed exclusion.
