# DD-008 report

This study is synthetic and exact. With accuracy `p`, common/common has social value `p`; a one-independent-source profile has `2p-p²-c`. Private deviation from common to independent is worthwhile only when `c < p(1-p)/2`, while the planner prefers one independent source when `c < p(1-p)`. Thus, in the open interval `p(1-p)/2 ≤ c < p(1-p)`, common/common is a source-choice equilibrium while one independent source is socially better. This is a bounded Common-Source Trap, not a universal theorem.
