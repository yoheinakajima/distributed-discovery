# DD-017 — Threshold Equilibrium Selection

This Program V4 study separates pure Nash occupancy, symmetric independent
mixing, pairwise blocking, and minimum-team coalition blocking in the exact
threshold-discovery game. It begins with a bounded small-game registry and does
not identify the DD-016 tied-mode rule with the full equilibrium correspondence.
