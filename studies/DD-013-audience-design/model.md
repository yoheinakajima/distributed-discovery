# Frozen model boundary

The study inherits DD-012's three-target, private-clue, shared-clue, equal-split
environment. Binding audiences determine who receives the shared clue;
voluntary audiences allow recipients to choose a registered action mode;
public role assignment requires an explicit compliance mechanism; and garbling
uses a separately registered lower-accuracy signal family. These institutions
are never treated as interchangeable.

The primary grid is `N=2,...,8` and
`p,q in {1/3,1/2,2/3,3/4,5/6}`. Symmetric garbling uses delivered accuracy
`g` from the same grid with `1/3<=g<=q`. Voluntary choice occurs ex ante between
the follow-shared and follow-private modes; signal-contingent rules are deferred
to DD-014. Public universal pooling pays each role `1/N` exactly when discovery
occurs and is evaluated as an ex-post budget-balanced implementation within
that registered mode class.
