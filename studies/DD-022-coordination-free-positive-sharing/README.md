improve decentralized discovery when anonymous agents respond to an equal-split
unit prize using only independent private randomization. Issue #150 and branch
`research/dd022-coordination-free-positive-sharing` own the sole substantive
lane. No result or claim exists before the clean immutable run and separate
claim audits.

completion gates are in `model.md`, `plan.md`, and `configs/registry.yml`.
