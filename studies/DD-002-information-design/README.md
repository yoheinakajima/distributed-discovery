# DD-002 — Information Design for Discovery

**Working subtitle:** When Better Public Signals Concentrate Search. Study whether more informative disclosure can reduce equilibrium discovery under a fixed strategic protocol.

The executable design, literature boundary, risks, and completion tests are specified in [brief.md](brief.md). The bounded deterministic-disclosure fixture is complete: all 15 policies, 37 posterior games, 256 global pure-equilibrium selections, and 45 strict refinement pairs are recorded in primary run `20260720T225848Z_DD-002_94607423_e29b1460ae`. Claims DD-C-0029 through DD-C-0031 report the complete census and exact selection-dependent reversal. Randomized disclosure remains outside this study's implemented scope.
