# DD-016 — Threshold Discovery

This registered Program V4 study asks how a discovery architecture should
allocate agents when a candidate opens only after a minimum team forms.
The working subtitle is *Coordination, Crowding, and Minimum Viable Teams*.

The primary scope is exact finite posterior-ranked search with one hidden
atomic target, one action per agent, deterministic threshold opening, no
execution failure, no value above the threshold, and equal prize splitting.
Planner allocations, independent mixed actions, correlated recommendations,
strategic choice, and selected equilibria remain distinct objects.
