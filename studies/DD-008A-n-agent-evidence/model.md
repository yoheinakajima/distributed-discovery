# Frozen model

There are `M=3` uniform targets and `N in {2,...,8}` agents. Each chooses a
free common source `C` or a costly independent source `I`, observes one
`p`-accurate clue, and directly searches its clue. All `C` users share one draw;
each `I` user has a conditionally independent draw. Successful searchers split
a unit prize equally, and social net value is union discovery minus `c` per
independent source. There is no disclosure, assignment, strategic action choice,
or real data.
