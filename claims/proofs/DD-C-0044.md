# DD-C-0044 — complete bounded heterogeneous moment census

For every one of the 839 canonical colored networks, the primary run stores the
complete first and pairwise scalar-report action moments and exact private
discovery. Grouping by the full 66-entry moment signature yields 163 matched
groups containing 485 networks. Exactly 111 groups contain more than one private-
discovery value.

The independent verifier reconstructs all 839 signatures and discovery values
without importing the primary heterogeneous model. Its labeled colored-object
orbit traversal confirms completeness, and its direct color-preserving
isomorphism check confirms the registry contains one representative per orbit.
It independently reproduces all 163 matched groups and 111 differing groups.

Weaker scalar summaries also have discovery-differing matches: mean pair agreement,
source HHI, mean source overlap, mean source accuracy, and exposure-weighted
accuracy each fail within the registered census. These counts describe one exact
finite class and do not imply that every pairwise diagnostic fails in all models.
