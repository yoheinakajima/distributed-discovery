# DD-C-0018 — deterministic sufficiency for the finite private team

## Claim

In the finite, static, common-payoff DD-001 model, ex-ante randomization cannot raise the private-team optimum above the best deterministic policy profile.

## Proof

There are finitely many deterministic maps from each finite signal set to the finite action set, hence finitely many deterministic labeled profiles \(f\). Any shared ex-ante randomized team strategy is a probability distribution \(\lambda\) over these profiles. Its discovery value is

\[
\sum_f \lambda_f G(f),
\]

which is a convex combination and is therefore no larger than \(\max_f G(f)\). Independent behavioral randomization by agents also induces a distribution over deterministic profiles by drawing every signal-contingent action ex ante, so the same argument applies. A maximizing deterministic profile exists because the set is finite.

This does not claim that strategic mixed equilibria are equivalent to pure equilibria, nor does it cover private randomization correlated with information unavailable at commitment time.

## Internal consistency check

The exhaustive optimizer searches only deterministic profiles. The proof justifies that restriction for the team optimum; it is not inferred from computational results.
