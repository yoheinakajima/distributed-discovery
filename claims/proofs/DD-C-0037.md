# DD-C-0037 — alignment-preserving count-budget upper bound

Fix a target \(t\). For role \(i\), let \(c_i(t)\) be the number of reports
mapped to action \(t\), and let \(d_i(t)\) indicate whether report \(t\) is
mapped to \(t\). With \(q=(1-p)/(M-1)\), the conditional success probability is

\[
s_i(t)=q c_i(t)+(p-q)d_i(t).
\]

The fixed-point indicator must be zero when \(c=0\), one when \(c=M\), and may
be either zero or one when \(0<c<M\). Define

\[
a(c)=\min_{d\in D(c)}\{1-qc-(p-q)d\}.
\]

For every actual policy column, \(1-s_i(t)\ge a(c_i(t))\). Conditional report
independence therefore gives

\[
\Pr(\text{all roles fail}\mid t)
=\prod_i(1-s_i(t))
\ge \prod_i a(c_i(t)).
\]

The first Bellman program computes the minimum right-hand side at each retained
joint-column resource:

\[
F_N(r)=\min_{c_1+\cdots+c_N=r}\prod_i a(c_i).
\]

Every deterministic policy maps exactly \(M\) reports, so its incoming counts
sum to \(M\). Consequently every \(N\)-role profile satisfies the necessary
global identity

\[
\sum_t\sum_i c_i(t)=MN.
\]

Dropping the separate per-role budgets, cross-target fixed-point compatibility,
and residual function/Hall feasibility enlarges the feasible set. Minimizing over
that superset yields

\[
\frac1M\sum_t\Pr(\text{all fail}\mid t)
\ge
\frac1M\min_{r_1+\cdots+r_M=MN}\sum_t F_N(r_t).
\]

Subtracting from one reverses the failure comparison and proves that the
relaxation value is an admissible discovery upper bound. The construction never
sorts roles independently inside a target: the full aligned count vector is
optimized jointly. The primary implementation emits both Bellman tables; the
separate verifier checks all recurrence inequalities and an attaining predecessor
for every state using exact fractions and no minimization call.

The relaxation need not be exact. It deliberately omits cross-target feasibility,
and the audited \(M=3,N=2,p=0\) case has upper bound \(1\) versus exact optimum
\(11/12\).
