# DD-C-0042 — exact color-preserving source census

The base model freezes three target labels, four searchers, nonisolated sources
and searchers, conditional source independence, plurality-winner-set reports,
independent-uniform private actions, and union discovery. A colored source is an
exact rational accuracy paired with one adjacency row. Canonicalization minimizes
the sequence of `(accuracy, row)` pairs over every source and searcher permutation;
therefore a relabeling moves a color with its source and cannot erase accuracy.

For the equal-baseline, modest, near-uninformative, and strong registered profiles,
direct enumeration gives 41,612 valid labeled colored objects and 671 orbits. The
single three-color expansion adds 12,966 labeled objects and 168 orbits, for 839
canonical networks total. A separate implementation enumerates the same labeled
objects and traverses each orbit using only adjacent source/searcher swaps. Its
nine profile/source-count orbit totals agree exactly with the primary canonicalizer.

Every registry representative is model-valid. Within every registered profile its
representatives are pairwise nonisomorphic under direct color-preserving search.
The 50 equal-accuracy two/three-source representatives reproduce every prior
homogeneous private-discovery and pairwise-agreement value. The census is complete
only for the profiles in ADR-0011.
