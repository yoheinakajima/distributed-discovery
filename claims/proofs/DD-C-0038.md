# DD-C-0038 — exact canonical private-team optimum

For \(M=16\), \(N=8\), and \(p=1/5\), direct clue-following is a feasible
deterministic profile. Each role fails to select the true target with probability
\(4/5\); conditional independence gives discovery

\[
1-(4/5)^8=\frac{325089}{390625}=0.83222784.
\]

By DD-C-0037, the alignment-preserving count-budget relaxation upper-bounds
every deterministic zero-communication role profile. The exact primary
certificate has local-column resource frontier \(F_8\), assigns resource eight
to each of the sixteen target columns, and certifies average failure

\[
\frac{65536}{390625}=(4/5)^8.
\]

Its discovery upper bound is therefore \(325089/390625\), equal to the feasible
direct value. The lower and upper bounds coincide, proving deterministic global
optimality.

An ex-ante randomized policy selects a deterministic role-policy profile before
the target and private reports are drawn. Its expected discovery is a convex
combination of deterministic-profile values, each at most
\(325089/390625\). Randomization therefore cannot improve the bound, while the
degenerate distribution on direct clue-following attains it. The randomized and
deterministic optima are equal.

The immutable run checks the certificate with an implementation that performs no
optimization: it verifies every exact Bellman lower inequality, one equality
witness for every state, the final resource witness, and the derived fraction.
It also rejects a certificate whose final Bellman value is replaced by zero.
