# DD-C-0039 — exact potential for the equal-split location game

Let the common posterior mass of location \(x\) be \(\mu_x\). At profile
\((a,b)\), a searcher choosing an unshared location receives \(\mu_x\), while a
searcher sharing the other player's location receives \(\mu_x/2\). Define

\[
\Phi(a,b)=
\begin{cases}
3\mu_a/2,&a=b,\\
\mu_a+\mu_b,&a\ne b.
\end{cases}
\]

Fix the other player's action \(b\) and compare a move from \(a\) to \(c\).
There are only the following occupancy changes:

- distinct to distinct: both payoff and potential change by \(\mu_c-\mu_a\);
- shared to distinct: both change by \(\mu_c-\mu_b/2\);
- distinct to shared: both change by \(\mu_b/2-\mu_a\).

The identity is symmetric between players, so every unilateral payoff difference
equals the corresponding potential difference. This is an exact potential game.

Every strict best-response move strictly raises \(\Phi\). The finite action-
profile set admits no infinite strictly increasing path, so the process terminates
at a state with no strict profitable deviation, equivalently a pure Nash
equilibrium. This argument does not assert termination when payoff-neutral moves
are allowed.
