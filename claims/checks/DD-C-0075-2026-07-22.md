# DD-C-0075 verification — 2026-07-22

Result: passed as a scoped `verified` theorem.

An incumbent's payoff depends only on its posterior mass, current occupancy,
and threshold viability. A unilateral move from j to k changes the deviator's
new occupancy to `n_k+1`; the effect of departure on j is payoff-irrelevant to
that agent. Comparing the incumbent payoff against every k is therefore both
necessary and sufficient for weak pure Nash equilibrium.

Clean run `20260722T024032Z_DD-017_033452f6_3d2c74fdfb` independently checks
all 3,728 occupancy states against 87,216 labeled profiles and reproduces every
equilibrium occupancy. Weak ties and zero-payoff subthreshold incumbents are
retained rather than silently refined away.
